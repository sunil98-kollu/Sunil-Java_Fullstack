   What is Local Cache ?
======================		

=> Local cache is stored within the application instance (in-memory).

=> It is fast because data is kept in the same JVM / process.


benfits
.....
data storage in application memory
performance faster 
each instance maintaon its own copy of data 
good for single node app

limitation
....................
not shared across mul instance 
data is lost we restart our app

..............

What is Distributed Cache ?	
============================	

=> Distributed cache is stored outside of the application, usually in a separate server or cluster.

Benfits 
.............
sharedacross all app instance
consistance data across all microservices 
data is preserved even if app restarts 
suitable for cloud , microservies ad app with scaling

limitation
..................
slightu slower than local cache (network calls)
req additional infrastructure

EX 
reddis gemfiles hazlaecast

When to use which cache ?	
=============================

==> Single server app ----------> Local cache is better

==> Microservices -------------> Distributed cache is better	 

==> App with multiple instances (scaling) ---------> Distributed cache is better

==> Max Performance (ultra fast) -----------> Local cache is better

==> Need shared state across instances --------> Distributed cache is better
