L → Liskov Substitution Principle (LSP)
.....................................

child class should replace by its parent class without baking functionalty

 Bad Example: Bird and Penguin
 ......................................
Parent
class Bird {

    public void fly() {
        System.out.println("Flying");
    }
}
Child
class Penguin extends Bird {

    @Override
    public void fly() {
        throw new UnsupportedOperationException(
                "Penguins cannot fly");
    }
}
Usage
Bird bird = new Penguin();
bird.fly();


 Good Example: Banking System
..............................
Parent Class

class Account {

    public void withdraw(double amount) {
        System.out.println("Withdraw successful");
    }
}
Child Classes
class SavingsAccount extends Account {

    @Override
    public void withdraw(double amount) {
        System.out.println("Savings Account Withdraw");
    }
}
class CurrentAccount extends Account {

    @Override
    public void withdraw(double amount) {
        System.out.println("Current Account Withdraw");
    }
}
Usage
Account account = new SavingsAccount();
account.withdraw(1000);

or

Account account = new CurrentAccount();
account.withdraw(1000);

Output
Savings Account Withdraw

Loan Kiosk

Bad Example
.....................
Low-Level Module
class MySQLDatabase {

    public void saveTransaction() {
        System.out.println("Transaction saved in MySQL");
    }
}

High-Level Module
class TransactionService {

    private MySQLDatabase db =
            new MySQLDatabase();

    public void processTransaction() {
        db.saveTransaction();
    }
}
Problem

Suppose tomorrow the bank wants to use MongoDB:

class MongoDatabase {
    public void saveTransaction() {}
}

Now you must modify:
Tightly coupled.


Good Example
.................. (Banking Application)
Abstraction


interface Database {

    void saveTransaction();
}

MySQL Implementation

class MySQLDatabase implements Database {

    public void saveTransaction() {
        System.out.println("Saved in MySQL");
    }
}

MongoDB Implementation

class MongoDatabase implements Database {

    public void saveTransaction() {
        System.out.println("Saved in MongoDB");
    }
}
Service
class TransactionService {

    private Database database;

    public TransactionService(Database database) {
        this.database = database;
    }

    public void processTransaction() {
        database.saveTransaction();
    }
}
